# sankofa-family

## installation

Use 'cd' to wwwroot folder, then do git clone:

```{r, engine='bash', count_lines}
git clone https://github.com/jasonkwh/sankofa-family
```

Afterwards, use the following commands:

```{r, engine='bash', count_lines}
cd sankofa-family
mv ./* ..
cd ..
rm -Rf sankofa-family
```
